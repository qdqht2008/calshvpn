Proxy SwitchyOmega 使用教程

什么是 SwitchyOmega

SwitchyOmega 是一款强大的浏览器代理管理扩展，支持 Chrome 和 Firefox。它可以帮助你轻松切换代理规则，控制哪些网站通过代理访问，哪些网站直接连接。

为什么需要 SwitchyOmega

节省流量：只让需要访问的国外网站走代理，国内网站直连，显著减少代理流量消耗。

解决内网限制：公司内网通常有访问限制，使用 SwitchyOmega 可以设置内网地址直连，代理只用于外部网络。

提升速度：直连国内网站速度更快，无需绕道代理服务器。

安装指南

Chrome 安装：

访问 Chrome 应用商店，搜索 "SwitchyOmega"，点击安装即可。

Firefox 安装：

访问 Firefox 附加组件商店，搜索 "SwitchyOmega"，点击安装即可。

基本配置步骤

第一步：创建代理配置文件

  1. 安装完成后，点击扩展图标，选择"选项"
  2. 点击"新建配置文件"
  3. 输入名称(如：Clash代理)，选择"代理配置文件"
  4. 配置代理服务器地址和端口(通常为 127.0.0.1:7890)
  5. 保存设置

第二步：创建自动切换规则

  1. 再次点击"新建配置文件"
  2. 输入名称(如：自动切换)，选择"自动切换配置文件"
  3. 在"默认代理"中选择刚才创建的代理配置
  4. 添加直连规则列表

推荐规则配置

直连规则（国内网站）

规则类型 示例
域名通配符 *.baidu.com, *.taobao.com, *.qq.com
完整域名 163.com, alipay.com, weibo.com
内网地址 192.168.*.*, 10.*.*.*, 172.16.*.*

代理规则（国外网站）

规则类型 示例
域名通配符 *.google.com, *.youtube.com, *.twitter.com
完整域名 netflix.com, openai.com, github.com
GFWList规则 【自动匹配】 - 内置的GFWList自动处理被墙网站

常见使用场景

场景一：内外网分离

在公司网络中，设置内网IP段直连：

  • 192.168.0.0/16 - 局域网
  • 10.0.0.0/8 - 私有网络
  • 172.16.0.0/12 - 私有网络
  • *.company.com - 公司内部网站

场景二：流媒体解锁

需要代理才能访问的网站：

  • Netflix、HBO、Disney+ 等流媒体
  • ChatGPT、Gemini 等AI服务
  • YouTube、Twitter、Facebook 等社交媒体

场景三：开发调试

设置特定域名直连用于本地开发：

  • localhost
  • 127.0.0.1
  • *.local

快捷切换

配置完成后，点击浏览器右上角的 SwitchyOmega 图标即可快速切换：

  • 代理模式：所有流量走代理
  • 直连模式：所有流量直连
  • 自动切换：根据规则自动选择

提示：建议日常使用"自动切换"模式，系统会自动判断哪些网站需要代理，哪些直连。
