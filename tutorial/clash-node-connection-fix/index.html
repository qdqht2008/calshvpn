Clash 节点连不上、延迟高怎么办？

代理无法上网、节点 timeout、测速全红的排查顺序

排查原则：先排除本机设置，再判断订阅和节点质量。不要一上来就重装客户端，很多问题只是系统代理、规则模式或 DNS 没开对。

一、先确认订阅是否更新成功

如果订阅没有更新成功，节点列表可能是旧的，甚至全是失效节点。先进入客户端订阅页面，手动更新一次订阅。

  • 更新失败：先看 Clash Verge 导入订阅失败排查。
  • 更新成功但节点全红：继续检查节点和线路。
  • 节点突然全部不可用：确认机场后台套餐是否过期或流量用尽。

二、检查代理模式和系统代理

Clash 客户端通常需要同时满足两个条件：选择了可用节点，并开启系统代理。如果只选了节点但没有打开系统代理，浏览器仍可能直连。

  1. 模式先选“规则”或“全局”测试。
  2. 打开系统代理开关。
  3. 选择一个香港、日本、新加坡或美国节点。
  4. 打开浏览器测试常用网站。

如果“全局”能用、“规则”不能用，问题可能在规则集；如果两者都不能用，继续排查节点和 DNS。

三、节点延迟高不等于一定不可用

客户端显示的延迟只是探测结果，不能完全代表实际体验。判断节点是否可用，建议同时看三个指标：

  • 测速能否返回：全部 timeout 通常说明节点或本地网络有问题。
  • 网页能否打开：延迟高但网页能打开，可能只是探测地址不稳定。
  • 晚高峰是否波动：晚高峰卡顿明显，通常和线路质量有关。

四、DNS 和 TUN 模式问题

如果浏览器能打开部分网站，但应用内无法联网，或域名解析异常，可以尝试关闭再开启 DNS/TUN 相关设置。不同客户端叫法不同，不建议同时开启多个代理软件。

提示：公司、校园网或公共 Wi-Fi 可能限制代理连接。换手机热点测试一次，可以快速判断是不是当前网络环境问题。

五、什么时候该换节点或换机场

如果只有单个节点不可用，换同地区其他节点即可。如果全部节点长期 timeout，且订阅更新正常，多半是机场线路质量或套餐状态问题。购买前可以先看 机场线路怎么选 和 机场避坑指南。

六、推荐排查顺序

  1. 手动更新订阅。
  2. 确认系统代理已开启。
  3. 切换全局模式测试。
  4. 换 3 个不同地区节点。
  5. 换手机热点测试。
  6. 确认机场后台套餐和流量状态。
