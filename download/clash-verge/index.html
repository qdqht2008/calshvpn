Clash Verge Rev 教程

如果您不知道 Clash Verge Rev 怎么用，请按下面的顺序完成基础配置：

第一步：下载、安装、启动

从上方下载对应系统的版本，安装并启动 Clash Verge。

第二步：准备好订阅链接

软件本身免费，但要实现代理功能，您需要添加有效的节点。如果您还没有 Clash 节点订阅链接，可以点此查看机场推荐免费试用。

第三步：导入配置链接

自动导入：点击 Clash 图标，稍等几秒即可自动导入配置文件。

手动导入：如果自动导入失败，点击"导入订阅"，复制订阅地址，粘贴到订阅地址输入栏，点击"下载"按钮。

第四步：打开系统代理开关

在"设置"中开启"系统代理"开关（确保没有红色报错提示），节点才会生效！

特别注意：务必开启系统代理开关，节点才会生效！

安装后遇到问题？订阅链接导入失败请看 Clash Verge 导入订阅失败排查；节点测速全红或代理无法上网请看 Clash 节点连接排查。

还没确定客户端？查看 Clash Verge Rev 和 Clash Meta 怎么选，按桌面端与 Android 平台区分使用场景。

此时您应该已经可以访问外部网络，可尝试打开 Google 进行测试。

常见问题

Clash Verge 还在更新吗？

是的。Clash Verge Rev 仍在维护和更新，基于 Clash Meta 内核，适合需要长期支持和新功能的 Windows、macOS、Linux 用户。

Clash Verge 安全吗？

Clash Verge 是开源客户端。建议只从 GitHub 官方项目或可信镜像下载，并避免安装来源不明的改包版本。

Windows、Mac、Linux 应该下载哪个版本？

Windows 用户通常下载 x64 setup 安装包；Apple Silicon Mac 用户下载 aarch64 dmg；Linux 用户可根据发行版选择 deb 包或前往 GitHub releases 页面选择对应版本。

Clash Verge 导入订阅失败怎么办？

先确认订阅链接完整、未过期、没有多余空格，并在浏览器里测试链接是否能打开。如果链接无效，需要回到机场后台重新复制或重置订阅。详细步骤可看 Clash Verge 导入订阅失败排查。

开启系统代理后仍然无法上网怎么办？

先确认已选择可用节点，再尝试切换全局模式、更新订阅、换节点和换网络。如果节点持续 timeout，需要继续排查线路质量、DNS 或机场套餐状态。详细步骤可看 Clash 节点连接排查。

Clash Verge 和 Clash for Windows 有什么区别？

主要区别：

  • 内核与更新：Clash Verge 使用较新的 Clash Meta 内核，功能更强，持续更新。而 Clash for Windows 已停止更新。
  • 功能特性：Clash Verge 提供更丰富的规则设置、更优的性能表现。
  • 用户界面：两者界面设计和交互体验有所不同。

建议：对于追求新功能和长期支持的用户，Clash Verge 是一个更推荐的选择。
