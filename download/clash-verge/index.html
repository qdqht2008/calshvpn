常见问题 ​

它还在更新吗？ ​

  1. 是的！这是一款积极维护和更新的客户端。它基于开源强大的 Clash Meta 内核，开发者会定期发布新版本以支持最新的代理协议、优化性能并修复已知问题。
  2. 您可以从官方 GitHub 仓库以获取最新版本信息。
  3. 根据我们的观察，近期新版本稳定性不佳，因此我们不建议您第一时间更新。
  4. 同样基于此原因，我们会观察一周左右，而不是在第一时间同步其官方最新版下载链接。

它安全吗？ ​

安全的！因为它本身是开源软件，其内核也是基于开源项目，代码接受社区监督。所以只要您从其 Github 官方渠道下载，并配合使用安全的节点订阅，风险很低!

它和 Clash for Windows 有什么主要区别？ ​

主要区别在于：

  1. 内核与更新：它使用较新的 Clash Meta 内核，功能更强，支持更多现代代理协议，并且持续更新。而 Clash for Windows 的官方版本已停止更新，仍基于旧版 Clash 内核。
  2. 功能特性：得益于 Meta 内核，它提供了更丰富的规则设置、更优的性能表现以及对新兴技术的支持。
  3. 用户界面：两者界面设计和交互体验有所不同。 总之，对于追求新功能和长期支持的用户，Clash Verge 是一个更推荐的选择。

应该选择它还是其它软件？ ​

如果您是 Windows 、macOS 或者 Linux 用户，并且：

  • 追求最新的代理协议支持和功能；
  • 希望客户端能持续获得更新和维护；
  • 偏好现代化的用户界面和体验。

那么它是你非常优秀的选择。它很好地平衡了功能强大性与易用性。

然而，如果您需要同时在电脑和安卓设备上使用，或者在界面美观上有更高的要求，那我们建议您对比下载中心提供的其它客户端，并重点考虑 FlClash 。
