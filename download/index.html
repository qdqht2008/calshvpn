使用问题 ​

怎么使用？ ​

您可在本下载中心的表格中，点击对应软件的“查看教程”链接。这些教程均采用图文并茂的形式，即使您是新手，也能在五分钟内快速上手。

按教程配置，仍无法使用 ​

如果您已经按照对应的教程步骤操作后，仍然无法访问外部网络，可能由多种因素导致。请不要着急，建议依次尝试以下步骤仔细排查，99%+情况下可以解决问题：

  1. 检查系统代理是否已打开；
  2. 检查配置文件是否已经成功导入；
  3. 检查节点是否可用
  4. 换个浏览器试试，只推荐用 Google Chrome、Edge 和 Safari，不要用一些既不好用也不安全的浏览器；
  5. 删除配置文件，重新导入一次；
  6. 卸载软件重装安装，再试一次；
  7. 卸载软件，下载安装其它软件,再试一次。

导入订阅失败怎么办？ ​

订阅链接是获取节点配置的关键。如果导入时遇到问题，例如提示:下载配置失败、get "https://xxxxx.xxxxxx" : EOF 或 长时间无响应，可以尝试以下方法：

  1. 删除配置文件，重新下载：先关闭系统代理，然后在配置页面删除现有的所有配置，再重新导入节点；再打开系统代理；
  2. 如果仍然失败：尤其是江苏、福建、河南的部分地区，下载配置的网址特别容易遭受干扰，建议向订阅地址的卖家求助。

怎么测试节点是否可用？ ​

大多数客户端都内置了节点连通性测试功能。这可以帮助您快速判断哪些节点当前可用且延迟较低。您可以点击节点页面的🛜 或⚡图标或者点击连通性测试之类的按钮。测试结果有数字的节点都是可用的，数字代表这些海外节点和设备的网络延迟；显示超时或失败的节点是不可用的。

测试节点全部失败？ ​

遇到这种情况，可以先尝试删除后重新导入配置文件再试试。如果结果仍然是节点全部超时，您应该求助卖家。

我应该选择哪个节点？ ​

  1. 通常，订阅服务商提供的默认配置文件中会自动选择或推荐合适的节点。对于日常使用，遵循默认设置往往能获得良好体验。
  2. 如果您是新手，在能正常访问网络的情况下，不建议更改设置和选项，避免发生异常情况导致无法正常联网。
  3. 当然，如果您已经是经验丰富的老手，或者有特定需求比如访问特定地区内容，可以考虑手动选择节点。

规则模式和分流规则是什么？ ​

  1. 代理工具通常提供多种工作模式，以适应不同的网络访问需求。
  2. “规则模式”（Rule Mode）是最常用也推荐新手使用的模式。它依赖于一套分流规则（规定了什么网站应该用什么节点访问），智能判断网络请求的目的地：访问国内网站和服务时直接连接，访问国外网站时则通过规则指定的代理节点。
  3. 这样既保证了访问国内服务的速度，又能顺畅连接外部网络，体验较好。

会影响国内网站的速度吗？ ​

一般不会。在推荐的“规则模式”下，客户端会智能判断网络请求的目的地。访问国内网站和服务时，流量会直接连接，不经过代理服务器，因此不会影响速度，也不会消耗您的代理流量。只有访问被规则定义为需要代理的国外网站时，才会通过所选节点。

不看外网时要关闭软件吗？ ​

  1. 电脑上：不需要，原因同上。
  2. 手机上：不嫌麻烦的话，可以关闭并退出代理软件，因为能给手机省点电😄

部分外网打不开？ ​

  1. 在代理页面，找到打不开的外网所属的分类，查看其选择的节点是否可用，如不可用，可点击切换到可用的节点。
  2. 如果不清楚其分类，大概率使用的是节点选择这个分类下所选中的节点。
  3. 如果仍然无法解决，可尝试使用全局代理模式。

怎么设置全局代理？ ​

  1. 全局代理是设备的所有网络包括国内软件，都用选择的节点进行代理。
  2. 在软件首页上方，默认模式是：规则，切换到左边的：全局。
  3. 全局模式的默认选项是：直连（DIRECT） ，也就是直接连接不代理。测速后选择任意可用节点，即可生效。
  4. 全局模式如果引起任何异常，请切换回规则模式。

使用后无法打开国内网站？ ​

  1. 使用代理软件后，连国内网站也无法打开，这种情况一般是发生在 Windows 系统。
  2. 首先您应该关闭并彻底退出软件：来到系统右下角托盘区域（包含已折叠图标），检查是否有猫咪图标。如果有，则右键点击图标，选择退出。
  3. 如果此时您仍然无法打开国内网站，依次选择：左下角 Windows 图标 - 右键 – 设置 – 网络和 Internet – 代理 – 手动设置代理，点击：编辑，关闭使用代理服务器的开关，点击保存。
  4. 现在，设备已经禁用了所有软件代理，应该可以直接连接国内网络了。

如果我更换了订阅服务或节点提供商，需要如何操作？ ​

如果您更换了订阅服务或节点提供商，您需要获取由新服务商提供的全新订阅链接。 操作步骤通常如下：

  1. 在您当前使用的代理客户端中，找到管理配置文件或订阅链接的入口（通常在“Profiles”、“配置”或“订阅”等菜单下）。
  2. 删除或禁用旧的、不再使用的订阅链接及其关联的配置文件。
  3. 添加新的订阅链接，并让客户端下载和解析新的配置文件。
  4. 选中新导入的配置文件，然后连接并测试节点是否正常工作。

相关技术知识 ​

Clash 内核是什么？ ​

  1. 内核是所有上网代理功能的核心。
  2. 它广泛地支持 Shadowsocks(SS)、ShadowsocksR(SSR)、Socks、Snell、V2Ray、Trojan 等代理协议；CFW 、CFA、ClashX 等知名软件都曾基于它；
  3. 尽管很遗憾，由于一些可以理解的原因，它停止更新了，但它拥有优秀的继承者。所以目前主流客户端都升级到了更新、更强大的 Clash Meta 内核。

Clash.Meta 内核是什么？ ​

  1. 简单来讲就是升级版。它站在巨人的肩膀上，继承了开源透明、高安全性的优秀传统，并且在功能上也是青出于蓝而胜于蓝，并且仍然在积极更新维护中。
  2. 目前主流的客户端比如 Clash Verge、Clash Meta for Android、FlClash 都基于 Clash Meta 内核(目前已经改名为 Mihomo)。

什么是TUN模式？我需要它吗？ ​

  1. TUN模式（或称虚拟网卡模式）是一种更深层次的系统级代理方式。它通过创建一个虚拟网络接口，能够捕获设备上几乎所有的网络流量，包括一些普通系统代理（如HTTP代理）可能无法接管的应用程序或游戏流量。
  2. 对于大多数用户的日常网页浏览、视频观看等需求，标准的系统代理模式已足够。如果您遇到特定应用程序（如图形化软件、某些游戏客户端）无法通过常规代理方式连接网络，或者需要代理非标准的HTTP/HTTPS流量时，可以尝试启用TUN模式。
  3. 请注意，TUN模式的配置和兼容性可能因操作系统和具体客户端而异。部分客户端（如 FlClash）对此有较好的内置支持，但有时可能需要额外的系统权限或配置。如果非必要，建议优先使用标准的系统代理模式。

什么是开源软件？ ​

开源软件是指其源代码（编写程序的原始指令）是公开的，任何人都可以查看、审查、修改和分发。这种透明度有助于发现和修复漏洞，并促进社区协作。

为什么要用开源代理软件？ ​

因为它们的运作方式是透明的，接受监督的，难以隐藏恶意行为。而代理本身是一种敏感私密的行为，所以选择开源的代理客户端很重要。

这些代理客户端是开源的吗？ ​

是的！本站推荐的所有代理软件（除了苹果手机），包括客户端和内核，全都是开源的，安全的！这也是为什么它们能成为最主流、最受欢迎的、通用的选择。
