配置教程

第一步：安装

下载 APK 文件，在手机上安装。如果提示安全警告，需要在设置中允许安装未知来源应用。

第二步：导入订阅

打开应用，点击添加订阅，粘贴您的订阅链接，下载配置。

第三步：启动代理

选择节点，点击启动代理按钮。

推荐理由：Clash Meta for Android 功能丰富，支持多种代理协议，界面友好。如果遇到 VPN 无法启动、订阅导入失败或安卓代理无法联网，请看 Android 配置失败排查。

常见问题

Clash Meta for Android 推荐使用吗？

推荐。Clash Meta for Android 基于 Clash Meta 内核，功能较新，适合需要 Android 代理客户端的用户。

Clash Meta for Android APK 安全吗？

建议只从官方 GitHub releases 或可信镜像下载 APK，不要安装来源不明的改包版本。安装前可确认文件名、版本号和下载来源。

Android 提示禁止安装 APK 怎么办？

需要在系统设置中允许当前浏览器或文件管理器安装未知来源应用。安装完成后，可以再关闭该权限。

订阅导入失败怎么办？

先确认订阅链接完整、未过期，并在浏览器里测试是否能打开。如果链接无效，需要回机场后台重新复制或重置订阅。详细步骤可看 Android 配置失败排查。

VPN 无法启动或经常断开怎么办？

先关闭其他 VPN 应用，重新授予系统 VPN 权限，再检查省电限制、后台保护和网络环境。部分 Android 系统会主动限制后台代理进程。

Clash Meta for Android 和 Clash for Android 有什么区别？

Clash Meta for Android 基于较新的 Clash Meta 内核，功能和协议支持更适合新用户。Clash for Android 属于经典旧客户端，官方版本已停止更新。
