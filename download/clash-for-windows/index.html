配置教程

第一步：下载安装

下载上面的版本，安装并启动 Clash for Windows。

第二步：获取订阅

如果您还没有订阅链接，请查看机场推荐获取。

第三步：导入配置

复制订阅链接，在 Clash for Windows 中粘贴订阅地址，下载并应用配置。

第四步：开启代理

在系统托盘或主界面开启系统代理开关。

提示：如果遇到节点 timeout、测速全红、开启系统代理后仍无法上网，请先看 Clash 节点连接排查教程。

常见问题

Clash for Windows 还更新吗？

Clash for Windows 官方版本已经停止更新，但旧版本仍可用于基础代理配置。如果需要长期维护和新功能，更建议选择 Clash Verge 或 FlClash。

Clash for Windows 现在还能用吗？

只要订阅链接有效、节点可用，并且系统代理正确开启，Clash for Windows 仍可正常使用。需要注意的是，停止更新的软件可能缺少新协议和后续兼容修复。

为什么下载或安装时会被 Windows 拦截？

Windows SmartScreen 或浏览器安全提示经常会拦截来自 GitHub 的可执行文件。请优先确认下载来源可信，再根据页面提示选择保存或仍要运行。

导入订阅失败怎么办？

先确认订阅链接没有过期、空格或复制不完整。如果浏览器也打不开订阅链接，需要回到机场后台重新复制或重置订阅。也可以参考 订阅导入失败排查教程。

节点 timeout 或代理无法上网怎么办？

先手动更新订阅，确认系统代理已开启，再切换全局模式和多个节点测试。如果仍然 timeout，继续排查网络环境、DNS、套餐状态和线路质量。详细步骤见 Clash 节点连接排查教程。

Clash for Windows 和 Clash Verge 选哪个？

新用户更推荐 Clash Verge，因为它仍在维护并支持较新的功能。Clash for Windows 更适合已经熟悉旧界面、只需要基础代理功能的用户。
