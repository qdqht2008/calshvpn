配置教程

第一步：安装

下载 DMG 文件，打开并将 ClashX Pro 拖入应用程序文件夹。

第二步：导入订阅

启动应用，在菜单栏点击图标，导入订阅，粘贴订阅链接。

第三步：启用代理

选择配置，点击启用系统代理。

提示：如果需要更现代的界面和更多功能，可以考虑 Clash Verge 或 FlClash。

ClashX Pro 常见问题

ClashX Pro 适合哪些 Mac 用户？

ClashX Pro 适合希望在 macOS 菜单栏快速导入 Clash 订阅、切换节点并启用系统代理的用户，尤其适合偏好轻量客户端的 Mac 用户。

ClashX Pro 怎么安装到 macOS？

下载 DMG 文件后打开安装包，将 ClashX Pro 拖入应用程序文件夹，再从应用程序或启动台打开。首次启动时按 macOS 提示授予必要权限。

ClashX Pro 怎么导入订阅链接？

启动 ClashX Pro 后点击菜单栏图标，进入配置或订阅管理入口，粘贴机场提供的 Clash 订阅链接并更新配置，然后选择可用节点。

ClashX Pro 如何开启系统代理？

导入配置并选择节点后，在菜单栏中启用系统代理。开启后，浏览器和多数系统网络请求会按 ClashX Pro 当前规则走代理。

ClashX Pro 连接不上怎么办？

先确认订阅可以正常更新，再检查是否已选择节点、是否启用系统代理、机场套餐是否仍有流量，以及当前节点是否可用。

ClashX Pro 和 Clash Verge Mac 版怎么选？

如果你偏好传统菜单栏客户端和轻量操作，可以选择 ClashX Pro；如果想要更现代的界面和更多桌面功能，可以尝试 Clash Verge。
