v2rayN 下载和使用教程

Windows 11、macOS、Linux 版本选择与订阅配置

v2rayN 是开源桌面代理客户端，官方项目支持 Windows、macOS 和 Linux，并可调用 Xray、sing-box 等内核。客户端本身不提供节点，使用前需要准备有效的服务器配置或订阅地址。

下载原则：只从 2dust/v2rayN 官方 GitHub 项目获取程序。发布文件会随版本变化，本页不固定某个版本或安装包。

Windows 11、macOS、Linux 怎么选

  • Windows：普通 Intel/AMD 电脑通常选择 64 位构建；Windows on ARM 设备选择 ARM64。
  • macOS：先区分 Apple Silicon 与 Intel 处理器，再按官方文件名选择。
  • Linux：按发行版和架构选择 deb、rpm 或压缩包，并查看官方发布说明中的依赖要求。

v2rayN 配置教程

第一步：下载并启动

从官方 Releases 页面下载与系统和架构匹配的文件。Windows 压缩包需要先完整解压，再从解压目录启动程序。

第二步：添加订阅

复制完整订阅地址，在订阅分组设置中新增订阅，保存后执行更新。不要把网页说明文字或换行一起复制进去。

第三步：选择节点

订阅更新后选择一个可用节点，可先测试延迟，再将其设为当前服务器。

第四步：开启系统代理

在 v2rayN 中开启系统代理，并选择适合当前需求的路由模式。不要同时开启多个会修改系统代理的客户端。

无法使用？订阅能更新但节点 timeout、浏览器无法联网时，可按 节点与系统代理排查顺序检查订阅、代理、DNS 和网络。

v2rayN 常见问题

v2rayN 只有 Windows 版吗？

不是。v2rayN 官方项目同时提供 Windows、macOS 和 Linux 构建，但中文用户最常使用的是 Windows 版。

v2rayN 应该下载哪个文件？

先确认操作系统和处理器架构。Windows 普通电脑通常选择 64 位构建，ARM 设备选择 ARM64；不确定时应查看官方发布文件说明。

v2rayN 怎么导入订阅？

在订阅分组设置中添加完整订阅地址，保存后更新订阅，再选择可用节点。不同版本的菜单名称可能略有差异。

v2rayN 开启后为什么浏览器不能上网？

先确认已选择可用节点，再检查系统代理是否开启。仍无法连接时，依次检查订阅状态、节点、路由模式、DNS 和当前网络。

v2rayN 更新后无法启动怎么办？

先查看官方发布说明和运行环境要求，并备份配置。若新版无法启动，可从官方 Releases 页面重新下载与系统架构匹配的完整文件。

其他客户端：返回 客户端下载中心，按平台比较 Clash Verge Rev、FlClash 和其他客户端。
