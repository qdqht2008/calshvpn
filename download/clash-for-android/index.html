常见问题 ​

CFA 还在更新吗？ ​

不，原作者已停止更新 Clash for Android。v2.5.12 是官方最后一个稳定版本。

下载的这个版本安全吗？ ​

若从本站提供的链接下载 Clash for Android APK，则软件本身是安全的。代理服务的安全性更多取决于您选择的 Clash 节点提供商。

CFA 和 CMFA 怎么选？ ​

  • Clash for Android：它的优点是极致简洁，适合老设备或者安卓电视。
  • Clash Meta for Android：推荐给绝大多数用户。它基于新内核，支持更多 Clash 节点协议和高级功能，仍然在持续更新，并且它的操作界面也跟 CFA 完全一致，是无缝替代。

CFA 支持哪些节点协议？ ​

支持 Clash 核心所支持的常见代理协议，如 Shadowsocks (SS), Vmess, Trojan 等。但如果是基于 Clash Meta 内核的新特性或协议的 Clash 节点，则可能无法完全兼容。
