配置教程

第一步：安装

下载 APK 文件，在手机上安装。需要在设置中允许安装未知来源应用。

第二步：导入订阅

打开应用，添加订阅，粘贴订阅链接，下载配置。

第三步：启动

选择节点，点击启动代理按钮。

说明：Clash for Android 官方版本已停止更新。如需最新功能，建议使用 Clash Meta for Android。如果遇到 VPN 无法启动、订阅导入失败或安卓代理无法联网，请看 Android 配置失败排查。

常见问题

Clash for Android 还更新吗？

Clash for Android 官方版本已经停止更新。旧版本仍可用于基础代理配置，但新用户更建议选择 Clash Meta for Android。

Clash for Android 现在还能用吗？

只要订阅链接有效、节点可用，并且 Android VPN 权限正常，Clash for Android 仍可使用。由于停止更新，它可能不支持部分新协议或新功能。

Android 提示禁止安装 APK 怎么办？

需要在系统设置中允许当前浏览器或文件管理器安装未知来源应用。安装完成后，可以关闭该权限以减少风险。

VPN 无法启动怎么办？

先关闭其他 VPN 或代理应用，重新授予系统 VPN 权限，再重启客户端。如果仍失败，可以重启手机并检查系统是否限制后台运行。

订阅导入失败怎么办？

先确认订阅链接完整、未过期，并在浏览器中测试是否能打开。如果链接无效，需要回到机场后台重新复制或重置订阅。详细步骤可看 Android 配置失败排查。

Clash for Android 和 Clash Meta for Android 选哪个？

新用户更推荐 Clash Meta for Android，因为它基于较新的 Clash Meta 内核并支持更多新功能。Clash for Android 更适合习惯旧版界面的用户。
